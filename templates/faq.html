FAQ

What is this site?

It is simply an improved interface for navigating the RosettaCode website. I love RosettaCode but I think it's constrained by the use of wikimedia backend which heavily determines overall structure of the website - great for browsing but not so great for structured learning and deeper exploration.

Two particular features that I think would be useful are the ability compare two languages side by side, and the ability filter out the more obscure languages and tasks and stick to what is most likely to be useful. This website is my attempt response.

Did you scrape RosettaCode?

Not exactly. I exported the programming tasks using the MediaWiki API, then parsed and rerendered it.

The links are broken!

Only the programming task links have been reimplemented. Any other links internal to RosettaCode will fail, sadly.

The rendering is broken!

The are still a lot of edge-cases that are yet to be dealt with - please fire me an email!
